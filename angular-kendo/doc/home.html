Angular directives for Kendo UI

DEPRECATED: This Kendo Labs repository has been deprecated. The Angular-Kendo bindings are now included in Kendo UI Core and will be available in the complete Kendo Q2 2014 (currently in beta).

This documentation still applies, with the notice that you must load AngularJS before Kendo UI in order for the directives to be created: 

<script src="jquery.min.js"></script>
<script src="angular.min.js"></script>
<script src="kendo.all.min.js"></script>

Kendo UI is a comprehensive HTML5/JavaScript framework for modern web and mobile app development. It contains many widgets for web, a core framework, a mobile suite, and a complete Data Visualization solution.

The Angular Kendo project aims to provide deep integration between Kendo UI Web/DataViz and AngularJS.

Easy as Pie!

<!-- markup -->
<div kendo-chart k-options="pie"></div>


// in controller:
$scope.pie = {
    title: {
        position: "bottom",
        text: "Share of Internet Population Growth"
    },
    seriesDefaults: {
        labels: {
            visible: true,
            background: "transparent",
            template: "#= category #: #= value#%"
        }
    },
